Конфигурации развертывания

Конфигурации для демонстраций, тестирования и пилотных проектов.

Конфигурация 1X

Для проведения демонстрации на ваших мощностях можно обойтись одной виртуальной машиной 

                    64-bit OS — семейство Linux OS, Ubuntu Server 20.04 или Astra-Linux Орел
                    CPU — 8 cores
                    RAM — 16 GB
                    Disk — 150 GB

Нам потребуется дистанционный доступ SSH, HTTP/HTTP(s), Web-Socket/Web-Socket(s) к ней. Порты 22/TCP, 80/TCP, 443/TCP, 8091/TCP.

Конфигурация 2X

Для определенного сорта пилотных проектов может быть рекомендована более устойчивая конфигурация из двух вычислительных машин. 

                    SAYMON CORE (программный комплекс "Центральный Пульт”)
                    64-bit OS — семейство Linux OS, Ubuntu Server 20.04 или Astra-Linux Орел
                    CPU — 8 cores
                    RAM — 16 GB
                    HDD — 100 GB

                    Под БД рекомендуется отдельную машину 
                    64-bit OS — семейство Linux OS, Ubuntu Server 20.04 или Astra-Linux Орел
                    CPU — 8 cores
                    RAM — 16 GB
                    SSD — 150 GB

Конфигурация 3X

Для проектов, в которых планируется использовать объем дополнительных модулей, рекомендуется вынести модули на отдельную машину. 

                    MODULES 
                    64-bit OS — семейство Linux OS, Ubuntu Server 20.04 или Astra-Linux Орел
                    CPU — 2 cores
                    RAM — 4 GB
                    HDD — 50 GB

                    SAYMON CORE
                    64-bit OS — семейство Linux OS, Ubuntu Server 20.04 или Astra-Linux Орел
                    CPU — 8 cores
                    RAM — 16 GB
                    HDD — 100 GB

                    БД
                    64-bit OS — семейство Linux OS, Ubuntu Server 20.04 или Astra-Linux Орел
                    CPU — 8 cores
                    RAM — 16 GB
                    SSD — 150 GB
